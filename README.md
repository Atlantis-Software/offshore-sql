# offshore-sql

[offshore](https://github.com/Atlantis-Software/offshore) adapter for sql databases.

## Installation

Install from NPM.

```bash
$ npm install offshore-sql
```
Install the database driver.

for Mysql
```bash
$ npm install mysql --save
```

Oracledb is in beta ...

more become later

## Offshore Configuration

```javascript
  config: {
    dbType: 'mysql',
    host: '127.0.0.1',
    user: 'root',
    password: '',
    database: 'offshoreSql'
  }
```
